GIT

git init - iniciar o GIT dentro de um diretório

git status - status atuais do git dentro do diretório, se existe arquivos que podem ser commitatos e quais arquivos possuem alteração e se estão salvas

git add NomeDoArquivo.ComExtensão - Salvando arquivo pro git, agora estando pronto para ser commitados

git commit -m "mensagem para commit" - commitando

git commit --amend -m"mensagem para alterar o ultimo commit" - altera a mensagem do ultimo commit

git log - histórico de commits

git reset --soft hasg_do_commit -

git reset --mixed hasg_do_commit -

git reset --hard hasg_do_commit -

git reflog - hitorico mais detalhado das alterações -

git remote add origin - conectar 2 diretórios git da maquina ao do GitHub

git branch -M main - Mudar nome de Master para Main

git push -u origin main - push responsável por enviar as alterações do local para o github

git remote remove origin - remover um repositório remoto de um diretório

rm -rf .git - apagar repositório git

mkdir nome_da_pasta - Criar um diretório

cd diretório - entrar em um diretório

touch nome do arquivo - criar um arquivo (bloco de texto extensão .md)
